# Calman's VimRC Configuration

## Prerequisites

Python LSP config:
```sh
pip install 'python-lsp-server[all]'
```

## Installation

Clone this repo:
```sh
git clone https://github.com/idocal/vimrc
ln -s ~/vimrc ~/.config/nvim
```

Install all plugins using [vim-plug](https://github.com/junegunn/vim-plug):
```
:PlugInstall
```
